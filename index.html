CSS Snippet cheatsheet

Ever having trouble recalling the exact syntax for your favorite CSS code? Select any snippet below and it will automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to create a flexbox row layout.

.row{
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use these properties to create a flexbox column layout.

.column{
    display: flex;
    flex-direction: column;
}

CSS Variables

Declare variables at the top of CSS file so that the variables could be reused throughout the file.

:root{
    --white: #FFF;
    --black: #000;
}

Media queries

Define breakpoints and put the media queries at the bottom of the stylesheet.

@media screen and (max-width: 768px){
   .card{
       width: 90%;
   }
}

Psuedo classes

Declare variables at the top of CSS file so that the variables could be reused throughout the file.

.card:hover,
.card:hover h2
{
    box-shadow: 0px 0px 15px var(--pink);
}
